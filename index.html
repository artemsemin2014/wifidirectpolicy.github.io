Политика конфиденциальности приложения WiFiDirect Рация

Приложение использует микрофон мобильного устройства для связи с другим устройством по протоколу связи WiFiDirect, связь с которым устанавливается пользователем. Микрофон используется при нажитии пользователем кнопки разговора. Приложение отправляет всю аудио информацию с микрофона только на другое мобильное устройство. Собранная информация не передаётся и не предоставляется третьим лицам. Скачивая приложение вы соглашаетесь с данной политикой конфиденциальности. Artem Semin 2017 (artemsemin2014@gmail.com)

Privacy policy for application WiFi Direct Walkie Talkie

The application uses the microphone of the user's device for connecting with another device which is set by the user via communication protocol WiFiDirect. The microphone is used when the user pushes the conversation button. The application sends all the audio information from the microphone to a different mobile device only. The collected information is not transfered to third parties. By downloading the app, you agree to this Privacy Policy. всё? Artem Semin 2017 (artemsemin2014@gmail.com)
